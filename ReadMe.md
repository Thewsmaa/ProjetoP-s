# Case E-commerce Olist — Relatório Executivo

Projeto desenvolvido como atividade da Pós-Graduação em Análise de Dados.

---

## Objetivo

Transformar dados transacionais do Brazilian E-Commerce Public Dataset by Olist em indicadores estratégicos para apoiar decisões de negócio.

A análise foi construída para responder perguntas relacionadas a:

- Crescimento comercial
- Eficiência logística
- Satisfação do cliente
- Desempenho dos sellers
- Oportunidades de expansão

---

## Ferramentas Utilizadas

- Power BI
- DAX
- Modelagem Dimensional (Esquema Estrela)
- GitHub

---

## Estrutura do Projeto

```text
powerbi/
→ Dashboard Power BI (.pbix)

reports/

```

## Modelagem dos Dados

Modelo desenvolvido utilizando arquitetura dimensional com tabela fato e dimensões auxiliares.

---

## Dashboard Executivo

### Página 1 — Crescimento e Receita

Indicadores:
- Receita Total
- Ticket Médio
- Pedidos Totais
- Evolução Mensal
- Top Categorias
  
---

### Página 2 — Logística e Operação

Indicadores:
- Tempo Médio de Entrega
- Percentual de Atrasos
- Frete Médio
- Impacto do atraso na avaliação

---

### Página 3 — Satisfação do Cliente

Indicadores:
- Nota Média
- Distribuição das Avaliações
- Categorias com maior satisfação
- Impacto do prazo de entrega

---

### Página 4 — Sellers e Oportunidades

Indicadores:
- Sellers Ativos
- Receita Média por Seller
- Top Sellers
- Receita por Estado

---

## Principais Resultados

- Receita total superior a R$ 1,36 Bi
- Crescimento consistente do volume de pedidos
- Tempo médio de entrega de 12 dias
- Taxa de atraso inferior a 8%
- Nota média geral acima de 4,0
- Concentração de receita em regiões específicas
